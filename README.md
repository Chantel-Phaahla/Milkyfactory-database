# 🍫 Milly’s Chocolate Factory – Database System

## 📌 Project Overview
This project is a database management system designed for Milly’s Chocolate Factory. The system was developed to manage customers, orders, branches, items, employees, accounts, and related business operations based on given business rules.

The main goal was to design a **fully normalized relational database (up to 3NF)** and implement it using Microsoft Access / SQL.
